How to install
=================

- Download the project and follow the instructions below:

```bash
1. git clone
2. cd dtmoney-ignite-rocketseat

```

- Install the dependencies and start the application:

```bash
3. yarn
4. yarn start
```
- or

```bash
3. npm install
4. npm run dev
```
